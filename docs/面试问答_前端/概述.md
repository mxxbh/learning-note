## 招聘要求

基本项：

- 学历，3 年以上前端开发经验
- 良好沟通、刻苦、敬业、有上进心、团队合作、爱专研、主动学习
- 精通 JS ( 函数闭包、继承、原型链 )
- 精通 CSS ( 动画 )
- 精通 HTML ( DOM )
- 熟练掌握网络协议 ( http、https、网络安全 )
- 熟练掌握 Web API ( BOM、客户端存储、Canvas2D )
- 熟悉兼容处理 ( 移动端、PC 端、常见浏览器 )
- 熟悉主流 MVVM 框架 ( React、Vue )
- 熟悉前端工程化工具 ( Gulp、Webpack )
- 理解效果优化 ( CSS 样式优化、页面加载性能优化、SEO )
- 理解代码重构与架构 ( 常用设计模式、好的编码习惯、微前端架构、模块化编程、函数式编程 )

加分项：

- NodeJS 开发经验
- Serverless、BI 开发经验
- LowCode 平台开发经验
- wap、小程序、快应用等开发经验
- 移动端 RN、Flutter 等项目开发经验
- github 开源社区活跃，有开源项目经验
