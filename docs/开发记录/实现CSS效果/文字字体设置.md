# 文字字体设置

通用字体定义：

```css
@font-face {
  font-family: Emoji;
  src: local('Apple Color Emoji'), local('Segoe UI Emoji'), local('Segoe UI Symbol'), local('Noto Color Emoji');
  unicode-range: U+1F000-1F644, U+203C-3299;
}
body {
  font-family: system-ui, -apple-system, Segoe UI, Roboto, Emoji, Helvetica, Arial, sans-serif;
}
```

数学公式字体定义：

```css
math {
  /** Cambria Math 是 Windows 操作系统中的数学字体 */
  /** Latin Modern Math 是 macOS 中的数学字体 */
  font-family: Cambria Math, Latin Modern Math;
}
```

响应式字体字号定义：

```css
html {
  font-size: 16px;
}
@media screen and (min-width: 375px) {
  html {
    /* 375px 作为 16px 基准，414px 宽度时正好对应 18px 的根字号大小 */
    font-size: calc(16px + 2 * (100vw - 375px) / 39);
  }
}
@media screen and (min-width: 414px) {
  html {
    /* 屏幕宽度从 414px 到 1000px，根字号大小累积增加 4px（18px-22px） */
    font-size: calc(18px + 4 * (100vw - 414px) / 586);
  }
}
@media screen and (min-width: 1000px) {
  html {
    /* 屏幕宽度从 1000px 往后每增加 100px，根字号大小就增加 0.5px */
    font-size: calc(22px + 5 * (100vw - 1000px) / 1000);
  }
}
```
