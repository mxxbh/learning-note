# 视频无法自动播放

在用户交互前，不能直接播放视频。

::: info 发生环境
安卓微信端、ios 微信端
:::

## 方式一

ios 微信端上可以利用 `WeixinJSBridgeReady` 事件触发视频播放。
安卓微信端暂时没有办法，只能增加交互，引导用户点击播放。

```js
document.addEventListener('WeixinJSBridgeReady', () => {
  window.WeixinJSBridge.invoke('getNetworkType', {}, () => {
    video.play();
  });
});
```
