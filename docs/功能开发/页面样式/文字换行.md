# 文字换行

响应式布局中，文字过长会自动换行，但默认会在最后几个字换行，导致词语被拆开，影响阅读。
期望是在特定字词位置换行。

## 方式一

使用软换行符 `\u200B` 分割文本。

<style module>
  .text {
    width: 100px;
    height: 100px;
    min-width: 80px;
    word-break: keep-all;
    overflow-wrap: break-word;
    box-shadow: 0 0 0 1px #333;
    resize: horizontal;
    overflow: hidden;
  }
</style>

<div class="docs-example">
  <div :class="$style.text">{{ text }}</div>
</div>

<script setup>
  import { ref } from 'vue';
  const text = "浏览器内核\u200B检测工具";
</script>
