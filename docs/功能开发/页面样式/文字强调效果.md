# 文字强调效果

<style module>
  .example1 {
    text-emphasis: '❤' red;
  }
  .example2 {
    -webkit-background-clip: text;
    background-clip: text;
    -webkit-text-fill-color: transparent;
    background-image: linear-gradient(to right, red, yellow, lime, aqua, blue, fuchsia);
    animation: hue 3s linear infinite;
  }
  @keyframes hue {
    from {
      filter: hue-rotate(0deg);
    }
    to {
      filter: hue-rotate(360deg);
    }
  }
  .example3 {
    font-size: 24px;
    background: linear-gradient(60deg, red, blue, green, black) center center / 100% 100%;
    background-attachment: fixed;
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
  }
</style>

## 方式一

强调文字。

<div class="docs-example">
  <p>这是一段用于<span :class="$style.example1">显示效果</span>的文字。</p>
</div>

```html
<style>
  .example {
    text-emphasis: "❤" red;
  }
</style>
<p>这是一段用于<span class="example">显示效果</span>的文字。</p>
```

## 方式二

文字颜色自动变换。

<div class="docs-example">
  <p :class="$style.example2">这是一段用于显示效果的文字。</p>
</div>

```html
<style>
  .example {
    -webkit-background-clip: text;
    background-clip: text;
    -webkit-text-fill-color: transparent;
    background-image: linear-gradient(to right, red, yellow, lime, aqua, blue, fuchsia);
    animation: hue 3s linear infinite;
  }
  @keyframes hue {
    from {
      filter: hue-rotate(0deg);
    }
    to {
      filter: hue-rotate(360deg);
    }
  }
</style>
<p class="example">这是一段用于显示效果的文字。</p>
```

## 方式三

文字颜色随滚动条滚动而变化。

使用 `background-attachment` 属性实现。
段落文字颜色设为透明，背景是从上到下的渐变且不随页面滚动。滚动页面时，文字发生滚动，透出后面的背景颜色。

<div class="docs-example">
  <p :class="$style.example3">这是一行随页面滚动自动变色的文字。</p>
</div>

```html
<style>
  p {
    font-size: 24px;
    background: linear-gradient(60deg, red, blue, green, black) center center / 100% 100%;
    background-attachment: fixed;
    -webkit-background-clip: text;
    background-clip: text;
    color: transparent;
  }
</style>
<p>这是一行随页面滚动自动变色的文字。</p>
```
